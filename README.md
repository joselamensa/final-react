Proyecto E-Commerce realizado durante la cursada de React en la plataforma de estudio Coderhouse.

# Elementos utilizados
- React.js
- React Router Dom
- React Bootstrap
- MDB Bootstrap

# Como iniciar el proyecto? 

1- Instalar dependencias:
npm install

2- Correr la app:
npm start
